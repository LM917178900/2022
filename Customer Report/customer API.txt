【template API】
1. 新增模板
2. 查询模板详情
3. 模板名称下拉框





【customer view API】
1. my report/product/version 级联下拉

2. search 
根据模板名称、产品id、版本号 查询报表

3. download
根据报表id 下载报表





【customer create API】
1. my report/product/status/version 下拉,(status/version有级联)
2. search
查询报表
进入页面查询：默认报表、默认产品，有draft 查询draft,没有实时查询
点击search：选择报表、选择产品，有draft 查询draft,没有实时查询

3. download
下载当前报表
4. refresh
同search
重新获取数据库数据用于页面展示（key update/photo 除外）

5. save 
没有保存过的数据，Save了就是Draft的状态了。
draft 将当前编辑的数据修改；
publish 将当前数据覆盖到draft 版本数据；

6. publish 
没有保存过的数据直接就Publish，那这时系统会自动保存一版Draft的数据和 publish 的数据。
Draft状态的数据，改为复制一份生成Published的版数据。 
publish 版的数据，生产有一版本新的数据；

7. delete
根据报表id ,逻辑删除报表，
相当于修改报表数据；

8. column config
9. time range selection
